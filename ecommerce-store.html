Project Details

E-Commerce Store

A fully functional online store with product listings, shopping cart, checkout flow, and order management — built with Node.js and Express on the back end.

Overview

This E-Commerce Store was built as a capstone project to demonstrate full-stack JavaScript development. The application provides a complete online shopping experience — from browsing products and managing a cart to placing orders. The back end is powered by Node.js with Express for routing and middleware, while MongoDB handles data persistence for products, users, and orders.

The front end uses EJS templates with Bootstrap for a clean, responsive shopping interface. Key features include user account management, product search and filtering by category, a persistent shopping cart, and a multi-step checkout flow with order confirmation.

Tech Stack
  • Node.js Server-side JavaScript runtime
  • Express Web framework and routing
  • MongoDB NoSQL document database
  • EJS Server-side template engine
  • Bootstrap Responsive front-end framework
Key Features

Architecture

The application follows an MVC pattern with Express handling routing and controller logic. Mongoose models define schemas for Products, Users, Carts, and Orders, each with built-in validation. Routes are split into modular files for products, cart management, authentication, and order processing.

Authentication uses bcrypt for password hashing and express-session for session management. The cart persists in the database so users can return and continue shopping. The checkout flow validates stock availability before confirming an order and updates inventory counts accordingly.
