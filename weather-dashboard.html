Project Details

Weather Dashboard

A dynamic weather application that fetches real-time data, displays 5-day forecasts, and features location-based search with an intuitive user interface.

Overview

This Weather Dashboard was built as a front-end project to practice working with external APIs and managing dynamic state in React. The application connects to the OpenWeatherMap API to provide real-time weather data for any city worldwide. Users can search for locations, view current conditions including temperature, humidity, and wind speed, and browse a detailed 5-day forecast with daily highs and lows.

The project showcases component-based architecture, asynchronous data fetching, responsive design, and conditional rendering based on API responses. Error handling ensures a smooth user experience even when network issues occur.

Tech Stack
  • React Component-based UI framework
  • JavaScript ES6+ application logic
  • CSS3 Custom responsive styling
  • REST API OpenWeatherMap integration
  • Vite Build tool and dev server
Key Features
  • Real-time weather data fetching
  • 5-day forecast with daily breakdown
  • City search with autocomplete
  • Dynamic weather icons and backgrounds
  • Responsive mobile-first design
